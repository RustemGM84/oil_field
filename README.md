
              
              ### База данных oil field (Нефтяное месторождение).


*Данная БД создавалась как первая часть учебного проекта. Использовалась СУБД MySQL. Следующая часть заключалась в получении данных с помощью Power Query, создание модели в Power Pivot,   
и визуализации в Power View. Эти операции были произведены на Power BI Desktop. Заключительной частью стала публикация отчётов и создание дашборда в Power BI Service.*

*Этот небольшой проект был создан для демонстрации потенциальному работодателю*
